Cipher Encryption & Decryption Application

Aplikasi ini adalah alat untuk melakukan enkripsi dan dekripsi teks menggunakan 3 cipher, yaitu:
1. Vigenere Cipher
2. Playfair Cipher
3. Hill Cipher

Langkah-langkah Penggunaan
1. Masukkan Teks:
Anda bisa mengetik teks langsung di area input atau mengunggah file teks dengan mengklik tombol "Unggah File".

2. Masukkan Kunci:
Masukkan kunci enkripsi atau dekripsi sesuai dengan cipher yang dipilih.

3. Pilih Cipher:
Pilih salah satu cipher dari menu (Vigenere, Playfair, atau Hill).

4. Pilih Aksi:
Klik tombol "Encrypt" untuk mengenkripsi teks, atau "decrypt" untuk mendekripsi teks.

5. Hasil:
Hasil dari proses Encrypt atau decrypt akan ditampilkan di area output di bawah.
